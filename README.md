# GameProject

A small Python/Pygame game project built as a learning exercise.

## Current Features

- Opens a Pygame window
- Draws a player square
- Moves the player using WASD
- Spawns one enemy
- Attacks the enemy with Space or left click while touching it
- Shows enemy health and defeat state

## Controls

- W = move up
- A = move left
- S = move down
- D = move right
- Space / left click = attack while touching the enemy
- R = respawn defeated enemy
- Esc = quit

## Run

```bash
python3 src/main.py
```

## Goal

Build a simple 2D game step by step, starting with movement, combat, enemies, and basic world interaction.
