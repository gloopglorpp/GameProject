# Game Ideas

## Core Vision

- [ ] Create a "moving painting" rather than a traditional game.
- [ ] Atmosphere-first design.
- [ ] Minimal dialogue and exposition.
- [ ] Environmental storytelling.
- [ ] Mystery over explanation.
- [ ] Emotional journey through changing landscapes.
- [ ] Large interconnected world with memorable locations.
- [ ] Strong inspiration from LIMBO's artistic philosophy.
- [ ] Warm orange fog and atmospheric lighting instead of black-and-white.

---

## Visual Style

- [ ] Black silhouette foreground objects.
- [ ] Abstract foggy backgrounds.
- [ ] Large shadow masses rather than detailed forests.
- [ ] Massive blurry pillars suggesting trees.
- [ ] Strong use of negative space.
- [ ] Light and shadow used as storytelling tools.
- [ ] Foreground silhouettes create depth.
- [ ] Player frequently walks behind objects.
- [ ] World feels larger than what is shown.
- [ ] Use ambiguity rather than realism.

---

## Atmosphere

- [ ] Heavy fog.
- [ ] Dynamic weather systems.
- [ ] Wind.
- [ ] Rain.
- [ ] Storms.
- [ ] Lightning.
- [ ] Deep sunsets.
- [ ] Distant thunder.
- [ ] Fireflies.
- [ ] Ambient environmental storytelling.
- [ ] Areas associated with specific moods and emotions.

---

## Weather Ideas

- [ ] Rain begins as a trickle.
- [ ] Rain slowly intensifies as player progresses deeper into an area.
- [ ] Thunder becomes more frequent.
- [ ] Storm eventually breaks into full downpour.
- [ ] Lightning reveals hidden elements.
- [ ] Weather affects atmosphere and gameplay.
- [ ] Weather acts as pacing mechanism.
- [ ] Rainbow after storm.

---

## Sound Design

- [ ] Heavy emphasis on sound design.
- [ ] Muffled low-pass filtered ambience in dense fog.
- [ ] Dynamic filtering based on location.
- [ ] Deep sub-bass thunder.
- [ ] Atmospheric drones.
- [ ] Recurring audio motifs.
- [ ] Environmental sounds carry emotional weight.
- [ ] Silence used intentionally.
- [ ] Sound changes with weather and biome.
- [ ] Soundscape behaves more like an electronic music arrangement.

---

## Recurring Bird

- [ ] Bird appears throughout the game.
- [ ] Bird observes major encounters.
- [ ] Bird flies away when approached.
- [ ] Bird may appear in impossible locations.
- [ ] Bird serves as recurring motif.
- [ ] No explicit explanation.
- [ ] No lore dump.
- [ ] Player speculation encouraged.
- [ ] Bird acts as observer or witness.
- [ ] Potential subtle audio motif associated with bird.

---

## Strange Objects

- [ ] Giant abstract structures.
- [ ] Impossible machinery.
- [ ] Massive buried wheels.
- [ ] Hanging cages.
- [ ] Strange towers.
- [ ] Objects larger than expected.
- [ ] Structures that feel symbolic rather than practical.
- [ ] Dreamlike architecture.
- [ ] Objects visible through fog.
- [ ] Objects partially obscured and unexplained.

---

## Environmental Encounters

- [ ] Old machine with glowing red light.
- [ ] Lever appears functional.
- [ ] Lever snaps when pulled.
- [ ] Light slowly fades.
- [ ] Encounter creates disappointment and mystery.
- [ ] Encounters often subvert expectations.
- [ ] Many moments should be memorable without combat.

---

## Exploration

- [ ] Large horizontal areas.
- [ ] Vertical transitions between regions.
- [ ] One-way drops.
- [ ] Slopes into new environments.
- [ ] Descents used to transition atmosphere.
- [ ] Camera follows terrain.
- [ ] Areas become distinct through mood and weather.

---

## Forest Area Ideas

- [ ] Swing hanging from giant tree.
- [ ] Ancient tree roots.
- [ ] Abandoned shack.
- [ ] Strange machinery.
- [ ] Lightning-revealed structures.
- [ ] Bird watching from distance.
- [ ] Heavy fog corridors.
- [ ] Silent clearings.

---

## Lightning Ideas

- [ ] World briefly illuminated.
- [ ] Hidden structures revealed.
- [ ] Creatures visible for only a fraction of a second.
- [ ] Different lightning overlays for different areas.
- [ ] Lightning acts as temporary source of information.
- [ ] Puzzle ideas involving flashes.

---

## Firefly Ideas

- [ ] Small group rather than swarm.
- [ ] Follow player loosely.
- [ ] Hover around head.
- [ ] Curious behaviour.
- [ ] Act like tiny creatures rather than particles.
- [ ] Warm glow.
- [ ] Personality and movement.

---

## Non-Combat Philosophy

- [ ] Avoid traditional combat where possible.
- [ ] Encounters based on tension and atmosphere.
- [ ] Distract.
- [ ] Avoid.
- [ ] Observe.
- [ ] Hide.
- [ ] Understand.
- [ ] Let the environment create danger.

---

## Emotional Themes

- [ ] Mystery.
- [ ] Wonder.
- [ ] Loneliness.
- [ ] Nostalgia.
- [ ] Curiosity.
- [ ] Transformation.
- [ ] Passage through different worlds.
- [ ] Becoming rather than winning.
- [ ] Symbolism over explanation.

---

## Technical Pipeline

- [ ] Photoshop is the source of truth.
- [ ] Large master PSD files.
- [ ] Atmosphere layer.
- [ ] Terrain layer.
- [ ] Foreground layer.
- [ ] Object layers.
- [ ] Export PNG assets.
- [ ] Use Pygame as runtime.
- [ ] Puppet Warp for animation.
- [ ] AI used for concepts and references.
- [ ] Editable source files preserved for everything.
